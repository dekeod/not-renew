# 不续签-拿满赔偿攻略
## 基本情况
### 入职
本人于2022年入职北京外企德科人力资源服务上海有限公司（后简称德科），被外派到深圳华为上班，合同签了4年，2026年到期

### 公司不续签
到期前几个月，收集了一下续签意愿，本人回复邮件：续签。

到期前1个月，HR通知被华为部门退回，询问是否要推荐至其他部门，本人拒绝。公司于是决定不续签

### 协商赔偿
到期前两周，周五，HR打电话谈赔偿，给的不满意，我说考虑下，发了《协商解除劳动合同协议》

## 协议的坑

### 一句话总结

赔偿金额少于法定，且附加了一堆附加额外条件，就是一个“不平等条约”，不能签署。

### 详细分析

1. 约定了总金额，没有约定金额构成，然后把所有的款项（包括但不限于加班费、奖金、公积金、未休年假等）列入此金额，签了之后可能会丧失继续追缴公积金、仲裁的权利。
2. N 赔偿金额 的是按 基本工资 x N，法定是 前12月月平均工资 x N。
3. 未休年假金额，给的是 基本工资 x 天数 x 2，法定是 工资 x 天数 x 3。
4. 公积金。协议中说乙方确认甲方已经依法缴纳公积金且放弃向任何行政部门提出申诉的权利。
   此项在印象中是不会被认定有效的，但是最近深圳公积金管理中心风向好像变了，开始承认个人与单位的协商有效了，可能会被认定为有效，从而拿不到少缴的公积金。
   参考资料：
   1. 《深圳市住房公积金管理委员会关于进一步规范我市住房公积金缴存提取等业务管理有关事项的通知》第七条 https://zjj.sz.gov.cn/ztfw/zfgjj/zcfg/gfxwj/content/post_12864290.html
   2. http://xhslink.com/o/4pPUBPHmJUU
6. 放弃仲裁、诉讼、向行政部门申诉的权利，这一条是格式条款，可能不会被认定，但是最好是删了。
7. 若乙方违反协议，甲方有权要求乙方返还所有款项。由于没有赔偿金额构成，这条可能会被认定有效。比如乙方追缴了公积金，甲方可能以违反协议起诉乙方，返还追缴到的公积金金额。

## 自己的权益（按川普的说法：手中的牌）

### 法定 N 赔偿（必须争取，仲裁必胜）
前12月均工资 x 4

### 未休年假三倍工资结算（必须争取，仲裁必胜）

### 公积金追缴（必须追缴，成功率接近100%）
深圳公积金缴纳：基数 x （5% ～ 12%）（德科是5%）
法定公积金基数计算规则：第1月可以不交公积金，第一年按第二月的工资作为基数，后面按上一年的月均税前工资作为基数
而德科的基数是基本工资

**工作4年，算出来会有 1万 ～ 几万 的差额**

### 周末加班工资差额（可以争取，仲裁有机会拿到）

目前德科的周末加班工资是以基本薪资作为基数计算的，而《深圳经济特区和谐劳动关系促进条例》第二十三条规定，劳动者加班工资计算基数为劳动合同约定的正常工作时间工资。

德科劳动合同约定了基本薪资、绩效工资，这二者极大概率会被认定为“劳动合同约定的正常工作时间工资”。

周末加班工资差额计算规则：绩效工资 / 21.75 x 加班天数 x 2

“如果加班多，这部分也能有 1万 ～ 几万 的差额”

法条：
1. 《深圳经济特区和谐劳动关系促进条例》第二十三条规定，劳动者加班工资计算基数为劳动合同约定的正常工作时间工资。
   https://hrss.sz.gov.cn/xxgk/zcfgjjd/zcfg/ldgx/content/post_1998353.html

### 社保追缴（对个人不划算，不建议追缴，可以作为谈判条件）

跟公积金类似，但是社保追缴后，个人还需补缴个人所交部分，还无法马上到手，不划算，不建议追缴。
ps：社保补缴具有强制性，协议约定无效。而且追缴长期有效，只要趁着公司倒闭前追缴，就能成功。

### 工作日加班工资（很难拿到，可以作为谈判条件）

基本不可能，公司有加班审批制度（应该进行了公示且无异议，有法律效应，这一条没有找到方法去确认），未审批加班视为自愿加班

## 谈判策略

### 分析自己的筹码（手中的牌）

可以根据上述章节，结合自己的情况，分析自己有哪些筹码，并做好分类与取舍。

分类：
1. 不能放弃的筹码（仲裁或追缴必胜）：法定 N 赔偿、未休年假三倍工资结算、公积金追缴
2. 可以权衡的筹码（预计很难谈下来，仲裁麻烦且有概率失败，金额小的话得不偿失）：周末加班工资差额   可以与HR协商修改协议，删掉赔偿金包含加班费、放弃仲裁的条款，后续通过仲裁拿到。
3. 可以放弃、用作退步条件的筹码（很难成功或者不划算）：社保追缴、工作日加班工资 
4. 额外手段：让HR相信你是个“硬茬”、表示懂法且不怕麻烦且要去仲裁、客户公司的任务交接、与客户公司领导的好关系、透露有亲戚或朋友当律师、把这次谈判与仲裁经历写出来起号做自媒体等等，可以自行发挥想象力。

### 谈判流程

#### 狮子大开口
不管能不呢成功，都要把所有筹码一起谈，其中一部分用作退步的条件

#### 坚持底线的基础上讨价还价
不放弃必须要拿到的筹码，逐步放弃本来就用作退步条件的筹码，可以权衡的筹码自行决断
可以上一些额外手段：让HR相信你是个“硬茬”、表示懂法且不怕麻烦且要去仲裁、客户公司的任务交接、与客户公司领导的好关系、透露有亲戚或朋友当律师、把这次谈判与仲裁经历写出来起号做自媒体

#### 做好谈崩去仲裁的准备
跟HR透露懂法且不怕麻烦且要去仲裁、有亲戚或朋友当律师，谈判很可能在这个阶段成功

要是不能成功，后续讨工资步骤：先去深圳投诉欠薪、上海（合同上的公司在上海）投诉欠薪，如果不能成功，就去仲裁

### 谈判不成功的大坑

#### 离职申请

最后一周的时候，公司可能会让走离职流程或者签离职申请书，千万不能直接走流程，不然可能会被认定为自愿离职

要以邮件要求公司发 不续签通知邮件 到你的私人邮箱后再走**非自愿离职**流程：**离职原因不能是个人原因，要明确写明是公司不续签**

#### 离职证明

离职证明上的离职原因不能写个人原因，否则让HR重开
